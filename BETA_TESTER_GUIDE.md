# 🧪 Guide pour les Testeurs Beta - JurisBot CI

Merci de participer au test de **JurisBot CI**! Votre feedback est crucial pour améliorer l'application.

## 🎯 Comment tester?

### 1. **Accéder à l'application**
```
Adresse: http://localhost:8501
```

### 2. **Naviguer dans l'onglet "Chat"**
- Posez vos questions sur le droit du travail ivoirien
- Lisez les réponses fournies par l'IA
- Observez les sources (articles cités)

### 3. **Laisser votre feedback**

#### Option A: Feedback Rapide
Cliquez sur l'un des boutons (immédiat):
- 👍 **Utile** — La réponse m'a bien aidé
- 👎 **Pas utile** — La réponse n'était pas pertinente
- 🚫 **Hallucination** — La réponse est erronée/contradictoire

#### Option B: Feedback Détaillé (Recommandé ⭐)
Remplissez le formulaire complet:

**📌 Précision juridique** (1-5)
- Est-ce que la réponse respecte la loi ivoirienne?
- Les articles cités sont-ils corrects?

**💬 Clarté** (1-5)
- Avez-vous compris la réponse?
- Est-ce exprimé en français clair?

**📖 Qualité des citations** (1-5)
- Les sources sont-elles exactes?
- Les numéros d'article sont-ils corrects?

**✅ Complétude** (1-5)
- La réponse couvre-t-elle complètement votre question?
- Y a-t-il des aspects manquants?

**💭 Commentaires**
Décrivez:
- Les erreurs détectées (avec détails)
- Ce qui manque dans la réponse
- Vos suggestions d'amélioration
- Anything unexpected or confusing

**📧 Email (optionnel)**
Pour que nous puissions vous suivre et répondre

## 🎯 Points clés à tester

### Précision juridique ⚖️
- [ ] Les articles cités correspondent-ils au code?
- [ ] Les règles énoncées sont-elles exactes?
- [ ] Y a-t-il des contradictions avec le contexte?

### Couverture légale 📚
- [ ] La réponse couvre-t-elle la question posée?
- [ ] Y a-t-il des cas particuliers manquants?
- [ ] Les exceptions sont-elles mentionnées?

### Clarté & Accessibilité 💡
- [ ] Le langage est-il accessible aux non-juristes?
- [ ] La structure est-elle facile à suivre?
- [ ] Les termes techniques sont-ils expliqués?

### Sources & Citations 📖
- [ ] Les articles sont-ils correctement numérotés?
- [ ] Les liens vers le texte sont-ils valides?
- [ ] Les dates de décrets sont-elles exactes?

---

## 🚨 Signaler des problèmes graves

Si vous découvrez:
- Une **hallucination** (réponse inventée)
- Une **contradiction** avec la loi
- Une **erreur critique** de citation

**Signalez-la via:**
1. Bouton 🚫 **Hallucination** dans l'app
2. Formulaire détaillé avec commentaires explicites
3. Email: racine.yao@gmail.com

---

## 📋 Checklist pour testeurs

Avant de conclure votre test:

- [ ] Testez au moins 5 questions différentes
- [ ] Évaluez les 4 critères (précision, clarté, citations, complétude)
- [ ] Laissez des commentaires constructifs
- [ ] Signalez les hallucinations détectées
- [ ] Vérifiez les statistiques dans l'onglet "Statistiques"

## 💬 Types de questions recommandées

### Questions simples
- "Quel est le délai de préavis pour un licenciement?"
- "Quelle est la durée légale du travail par semaine?"
- "Les heures supplémentaires sont-elles obligatoires?"

### Questions complexes
- "Comment fonctionne le télétravail en Côte d'Ivoire?"
- "Quels sont les droits d'une femme enceinte?"
- "Quelle est la procédure de grève?"

### Questions limites
- "Est-ce que les apprentis ont les mêmes droits?"
- "Qui finance les formations professionnelles?"
- "Comment se calcule l'indemnité de licenciement?"
