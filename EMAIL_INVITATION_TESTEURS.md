Un assistant IA qui répond à vos questions sur:
- Le Code du Travail ivoirien

- Les dispositions légales sur le télétravail

Toutes les réponses sont citées avec les articles de loi exacts.

2. Posez vos questions sur le droit du travail

3. Évaluez les réponses avec le système de feedback

4. Consultez le guide complet:
   📚 [GUIDE_UTILISATEUR.md](https://github.com/yyracine/jurisbot_ci/blob/main/GUIDE_UTILISATEUR.md)

⭐ **Votre Rôle est Important!**
Vos retours nous aident à:
✅ Améliorer la précision du système
✅ Détecter les hallucinations
✅ Rendre l'app plus utile
✅ Valider la qualité juridique

📝 **Feedback Simplifié**
Après chaque réponse, cliquez simplement:
- 👍 Utile
- 👎 Pas utile
- 🚫 Hallucination

Pour un feedback plus détaillé (recommandé), remplissez le formulaire avec vos observations.

- 📧 Contact: racine.yao@gmail.com

- Commentaires libres
- Email (optionnel, pour suivi)

Plus vous donnez de détails, mieux on peut améliorer!
